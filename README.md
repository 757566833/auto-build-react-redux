auto-build-react-redux
=========================

## Installation

```
npm install auto-build-react-redux -g
```
脚手架里面包含了指定版本的react，如果你清晰的知道react的版本依赖 你可以remove react  并install react 你想要的版本（同样适用于redux等全家桶）
